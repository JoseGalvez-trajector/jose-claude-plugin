# Greeting Skill

Provide warm, personalized greetings tailored to the context and recipient.

## When This Skill Applies

Activate when the user wants to:
- Greet a person or group
- Write an introduction message
- Start a conversation with a friendly opening
- Create a welcome message

## Guidelines

1. **Use the recipient's name** when provided — personalization makes greetings more meaningful.
2. **Match the tone** to the context: casual for friends, professional for work introductions.
3. **Keep it concise** — a greeting sets the tone but shouldn't overwhelm.
4. **Include context** when relevant (e.g., "Welcome to the team!" for a new hire).

## Examples

### Casual
> "Hey Alex! Great to meet you — looking forward to working together."

### Professional
> "Hello Dr. Smith, thank you for taking the time to connect. I'm excited to discuss the project with you."

### Welcome message
> "Welcome aboard, Jordan! We're thrilled to have you join the team. Don't hesitate to reach out if you need anything."
